# Prepare Interview Plan

Generate comprehensive interview plans for web developer candidates. Create both generic templates and candidate-specific customized plans.

## Two-Plan Output

When asked to prepare an interview plan, deliver:

1. **Generic Interview Template**
   - Reusable framework for all candidates
   - Standard question bank by topic
   - Timing guidance
   - Coding challenge difficulty levels

2. **Candidate-Specific Customized Plan**
   - Adapted to candidate's experience level and background
   - Focused on relevant skill gaps or strengths to verify
   - Tailored question difficulty
   - Adjusted timing based on role seniority

## Generic Template Structure

```
INTERVIEW PLAN - [ROLE_TITLE]
Duration: 60 minutes total

1. INTRODUCTION & BACKGROUND (5-10 min)
   - Welcome and role overview
   - Candidate background and experience
   - Interest in position

2. TECHNICAL Q&A (25-30 min)
   - Frontend Questions (5-7 questions)
   - Backend/Full-stack Questions (5-7 questions)
   - System Design Questions (2-3 questions)
   - Follow-up probes for depth

3. CODING CHALLENGE (20-25 min)
   - Problem statement
   - Expected solution approach
   - Evaluation criteria
   - Optimization discussion

4. WRAP-UP (5 min)
   - Candidate questions
   - Next steps
   - Thank you
```

## Customization Factors

Adapt plans based on:
- **Experience Level**: Junior, Mid-level, Senior, Staff
- **Specialization**: Frontend-heavy, Backend-focused, Full-stack
- **Background**: Self-taught, bootcamp, CS degree, career-changer
- **Current Skills**: Languages known, frameworks used, gaps to assess

## Question Bank Categories

For each interview, suggest:
- **3-4 foundational questions** (basic understanding)
- **3-4 intermediate questions** (practical experience)
- **2-3 advanced questions** (depth and architecture thinking)
- **Follow-ups** to probe deeper on each topic

## Coding Challenge Selection

Choose challenges based on:
- Role requirements (frontend focus vs. full-stack)
- Experience level (difficulty scaling)
- Key skills to assess (data structures, algorithms, API design)
- Time constraints (solvable in 20-25 minutes)

## Output Format

Present plans as:
1. **GENERIC TEMPLATE** - Immediately usable for any candidate
2. **CUSTOMIZED PLAN** - For the specific candidate mentioned

Include:
- Time allocations
- Specific questions tailored to their level
- Coding challenge recommendation
- Evaluation rubric for scoring
