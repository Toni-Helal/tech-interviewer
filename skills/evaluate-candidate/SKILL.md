# Evaluate Candidate Performance

Create comprehensive candidate evaluations after interviews. Assess technical knowledge, soft skills, and provide hiring recommendations.

## Evaluation Framework

Score candidates across these dimensions:

### 1. Technical Knowledge (0-100)
- **Frontend** (React, Vue, JavaScript, CSS): depth, patterns, best practices
- **Backend/Full-stack** (Node.js, databases, APIs): architecture, scalability
- **System Design**: ability to design systems, trade-off analysis

Scoring:
- 80-100: Expert level, designs systems, explains trade-offs deeply
- 60-79: Solid practitioner, knows patterns, occasional gaps
- 40-59: Foundational knowledge, needs guidance on complex problems
- 20-39: Struggling with fundamentals, significant gaps
- 0-19: Insufficient knowledge for role

### 2. Problem-Solving Approach (0-100)
- How they approach unfamiliar problems
- Ask clarifying questions
- Breaks down complexity
- Considers edge cases
- Iterative improvement

### 3. Code Quality (0-100)
- Readability and naming conventions
- Efficiency and algorithmic thinking
- Handling edge cases
- Testing mindset
- Error handling

### 4. Communication (0-100)
- Explains thinking clearly
- Listens and incorporates feedback
- Asks clarifying questions
- Can articulate trade-offs
- Comfortable with ambiguity

### 5. Soft Skills & Culture Fit (0-100)
- Enthusiasm for learning
- Collaboration signals
- Curiosity about challenges
- Humility and openness
- Long-term alignment with team

## Scoring Guidelines

**STRONG YES (85+)**: Exceptional candidate, hire immediately
**YES (70-84)**: Strong candidate, move to next stage
**MAYBE (55-69)**: Solid basics but gaps remain, conduct deeper assessment
**NO (40-54)**: Significant gaps, not suitable now, revisit in 12+ months
**STRONG NO (0-39)**: Major concerns across dimensions, not recommended

## Key Evaluation Rules

- Base scores on evidence from interview
- Note specific examples for each dimension
- Consider role requirements when weighting scores
- Separate "can learn" vs. "fundamental gaps"
- Account for interview pressure and nerves
- Compare fairly to role level expectations
